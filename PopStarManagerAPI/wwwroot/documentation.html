PopStarManagerAPI Documentation

This document provides an overview of the available API endpoints and their usage.

Base URL

The base URL for the API is: http://localhost:5071/api/popstar

Endpoints

GET /api/popstar

Retrieves all musicians.

Response:
[
  {
    "id": 1,
    "name": "Ariana Grande",
    "category": "Pop",
    "pricePerHour": 16,
    "imageUrl": "/images/Ariana.jpg"
  },
  ...
]

GET /api/popstar/{id}

Retrieves a musician by ID.

Request:
GET /api/popstar/1

Response:
{
  "id": 1,
  "name": "Ariana Grande",
  "category": "Pop",
  "pricePerHour": 16,
  "imageUrl": "/images/Ariana.jpg"
}

GET /api/popstar/search?name={name}

Searches for musicians whose names contain the provided query string.

Request:
GET /api/popstar/search?name=Ariana

Response:
[
  {
    "id": 1,
    "name": "Ariana Grande",
    "category": "Pop",
    "pricePerHour": 16,
    "imageUrl": "/images/Ariana.jpg"
  }
]

POST /api/popstar

Creates a new musician. This endpoint accepts multipart/form-data with the following fields:

  • Name (string, required)
  • Category (string, required)
  • PricePerHour (decimal, required)
  • ImageFile (file, optional)
Request (multipart/form-data):
Name: Ariana Grande
Category: Pop
PricePerHour: 16
ImageFile: [File]

Response:
{
  "id": 2,
  "name": "Ariana Grande",
  "category": "Pop",
  "pricePerHour": 16,
  "imageUrl": "/images/Ariana.jpg"
}

PUT /api/popstar/{id}

Updates an existing musician. Accepts a JSON body with updated fields.

Request:
PUT /api/popstar/1
Content-Type: application/json

{
  "id": 1,
  "name": "Ariana Grande Updated",
  "category": "Pop",
  "pricePerHour": 20,
  "imageUrl": "/images/Ariana.jpg"
}

Response:
204 No Content

DELETE /api/popstar/{id}

Deletes a musician by ID.

Request:
DELETE /api/popstar/1

Response:
204 No Content

Notes

If uploading an image, ensure the request uses multipart/form-data encoding. Uploaded images are stored in the wwwroot/images folder and served as static files.

For further questions, please contact the API developer.
